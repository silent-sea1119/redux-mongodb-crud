# redux-mongodb-crud
MongoDB, Express, React.JS, Node.JS

## Download : 
```
git clone https://github.com/WDcontrol/redux-mongodb-crud.git
cd redux-mongodb-crud
```

## Launching API : 
```
cd back && npm install
npm run serve
```

## Launching server client :
```
cd front && npm install
npm run start
```
